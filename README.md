# 3D-Graphics-Rendering-Vulkan

## 编译环境和依赖
* Windows
* Visual Studio 2022
* Vulkan SDK 1.4.313
* cmake 3.31
* conan 2
* C++ 20
* Python 3.6+

## 第三方库
* glfw 3.4
* spdlog 1.15.3
* assimp 5.4.3
* stb cci.20230920
* imgui 1.92.0
* implot
* ktx 4.3.2

## 特性
* Vulkan-HPP
* Vma
* Dynamic Rendering
* Slang
